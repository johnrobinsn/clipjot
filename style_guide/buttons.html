ClipJot Style Guide / Buttons

Buttons

Button styles and variants for all ClipJot platforms.

Primary Buttons

Use for main actions like "Save", "Submit", "Login".

<button class="btn btn-primary">Save Bookmark</button>

With Icons

Loading State

Secondary / Outline Buttons

For secondary actions like "Cancel", "Edit", "View More".

<button class="btn btn-outline btn-primary">Cancel</button>

Ghost / Text Buttons

Subtle buttons for tertiary actions, toolbars, or icon-only actions.

<button class="btn btn-ghost">Dismiss</button>

Destructive Buttons

For delete, remove, or other destructive actions.

<button class="btn btn-error">Delete</button>
