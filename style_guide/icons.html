Current Platform Comparison

Platform Current Approach Recommendation
WebUI Emoji icons (e.g., bookmark emoji) Migrate to Heroicons SVG
Chrome Extension Heroicons SVG (outline) Keep as is (reference implementation)
Android Material Vector Icons Convert Heroicons to vector drawables

Why Heroicons?

  • Clean, consistent outline style
  • MIT licensed, free for commercial use
  • Available as SVG (easy to use in web)
  • Can be converted to Android vector drawables
  • Maintained by Tailwind CSS team
  • Both outline and solid variants available
